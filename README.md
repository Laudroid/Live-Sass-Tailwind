# Cours Sass Tailwind

## Objectifs de la formation

- Structurer un CSS scalable avec Sass
- Comprendre les limites du CSS traditionnel et y répondre
- Mettre en place une architecture (7-1 pattern, modularité)
- Utiliser efficacement Tailwind CSS
- Comparer Sass vs Tailwind vs CSS natif
- Construire une UI moderne, responsive et maintenable
- Mettre en place une stratégie CSS dans un projet réel
## Séance 1 : Introduction et limites du CSS traditionnel

**Durée :** 3h

### Objectifs pédagogiques

- Comprendre les limites du CSS traditionnel
- Introduction à Sass et ses avantages
- Découverte de Tailwind CSS
- Comparaison rapide Sass vs Tailwind vs CSS natif
### Contenus

#### Limites du CSS traditionnel

- Spécificité, cascade, et problèmes de maintenance
- Redondance et duplication des styles
- Difficultés avec la modularité et la réutilisabilité
#### Introduction à Sass

- Historique et rôle de Sass
- Fonctionnalités principales : variables, nesting, mixins, fonctions
- Exemple simple d'utilisation
#### Introduction à Tailwind CSS

- Philosophie utility-first
- Principales classes et configuration
- Avantages et inconvénients de Tailwind
#### Comparaison Sass vs Tailwind vs CSS natif

- Quand choisir Sass, Tailwind ou CSS natif
- Cas d'usage et performances
- Interopérabilité et intégration dans les projets
## Séance 2 : Architecture et mise en place avec Sass

**Durée :** 4h

### Objectifs pédagogiques

- Mettre en place une architecture scalable avec Sass (7-1 pattern)
- Modulariser le code Sass
- Utiliser les fonctionnalités avancées de Sass
### Contenus

#### Présentation du 7-1 pattern

- Organisation des fichiers : base, layout, components, utilities, vendors, themes, abstracts
- Avantages pour la maintenabilité et scalabilité
#### Modularité avec Sass

- Partial files et import
- Mixins, fonctions personnalisées
- Gestion des thèmes et variables globales
#### Fonctionnalités avancées de Sass

- Control directives (if, for, each, while)
- Fonctions personnalisées avancées
- Extensions avec @extend et placeholders
## Séance 3 : Maitriser Tailwind CSS et intégration avancée

**Durée :** 4h

### Objectifs pédagogiques

- Configurer Tailwind CSS dans un projet
- Personnaliser la configuration Tailwind
- Utiliser les plugins Tailwind
- Stratégies pour UI responsive avec Tailwind
### Contenus

#### Installation et configuration de Tailwind CSS

- Installation avec npm/yarn
- Configuration du fichier tailwind.config.js
- Purge des classes inutilisées
#### Personnalisation de Tailwind

- Définition de thèmes et palettes de couleurs
- Ajout de fonts et custom utilities
- Gestion des breakpoints
#### Utilisation des plugins Tailwind

- Présentation et installation des plugins courants
- Création de plugins personnalisés
#### UI responsive et Tailwind

- Techniques pour rendre un site responsive
- Utilisation des variants responsive et hover
- Gestion des états et animations
## Séance 4 : Projet pratique et stratégies CSS modernes

**Durée :** 4h

### Objectifs pédagogiques

- Appliquer une stratégie CSS dans un projet réel
- Construire une UI moderne, responsive et maintenable
- Comparer les méthodologies dans un contexte réel
- Réflexion sur l'avenir du CSS avec Sass et Tailwind
### Contenus

#### Mise en place d'un projet complet

- Choix de la méthodologie (Sass, Tailwind, mixte)
- Architecture du projet et organisation des fichiers
#### Construction de l'UI

- Développement des composants UI
- Gestion de la responsivité et des thèmes
- Tests et maintenance
#### Comparaison et bilan

- Points forts et limites de Sass et Tailwind
- Choix adaptés selon le contexte du projet
- Perspectives d'évolution et bonnes pratiques
